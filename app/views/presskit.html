What is this?

A website where gamers can find new Friends to play multiplayer games with! It supports a wide variety of platforms ranging from 3DS to PlayStation4, from PsVita to Steam. Your friends sometimes don't have time to play online games with you, or don't own the games you want to play? This website can help!

Core Features

  • Filter all posts by platform, name of the game, timezone and language.
  • Easily set up a new PlayDate - no signup required.
  • NO SIGNUP REQUIRED - all messaging is done via E-Mail.

How does it work?

  1. You know which game you want to play? Search all PlayDates by filtering - select you platform, then start typing your game title
  2. Found a match? Start messaging by clicking the message button
  3. No fitting PlayDates available? Create your own! Just click the add PlayDate button

Why?

January 2015 I (Alex) was surfing reddit and stumbled upon /r/playdate. Lots of gamers where looking for gaming partners, but reddits search functionality as well as the unstructured posts on the subreddit made finding matches really hard. PlayDate was born that day. I wanted to make it easier for gamers to find new friends. After about 3 weeks of working on and off on the idea find-playdate.com goes online for the first time.
